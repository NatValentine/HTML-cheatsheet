Paragraphs, inline elements and block elements

There are two types of tags (elements), Inline and Block.

Inline elements do not start a new line and take only the necessary width. For example, span, images, links and some styling such as strong or emphasize.

Block elements, meanwhile, start on a new line and take up the whole width. Some examples are headings, div, paragraphs (like this one) and forms.

Tags can have attributes, they provide information about an element. They are placed within the open tag. A good example of this is the link tag.
